# telemetry-in-midair-rs

## ESP32-C6
`ESP32-C6-MINI-1U-H4`
`4MB Flash`

| Pin | Function |
|-|-|
| I03 | LED D2 |
| IO2 | PWR EN GPS/Radio (AP2112K-3.3) |
| IO4 | RX/GPIO |
| IO5 | TX/GPIO |
| RXD0 | WIO-E5 PA2 |
| TXD0 | WIO-E5 PA3 |

*Boot pad on back*

| PC1 | I2C SDA (JST SH) |

| PC1 |  |
